source code 

Creating a LineList object.

Any database from which a linelist is requested requires an entry for given molecule and database in path/MOLECULE_DATABASE.dat .

Parameters:
  • molecule (Molecule()) - The molecule for which LineList is requested
  • x_min (float) - Minimum value for reading data
  • x_max (float) - Maximum value for reading data
  • x_unit (string) - Unit of wave/freq (micron,ghz,mhz,hz)

    (default: 'micron')

  • cdms (bool) - read molecule data from CDMS catalog

    (default: 0)

  • jpl (bool) - read molecule data from JPL catalog

    (default: 0)

  • lamda (bool) - read molecule data from LAMDA catalog

    (default: 0)

  • min_strength (float) - if None all are included, else only lines with strengths above this value are included (log scale, nm2*Mhz)

    (default: None)

  • max_exc (float) - if None all are included, else only lines with excitation energies below this value are included (cm-1)

    (default: None)

  • path (string) - path where the line lists have been stored, names of files are molecule_DATABASE.dat

    (default: '~/LineLists/')

  • include_extra (bool) - include extra information such as integrated line intensity and excitation energy

    (default: 0)
