buildPhotometry(star_name, fn='Photometric_IvS', psuffix='Raw/IvS_SEDTool/', remove=[], dp='/home/robinl/Data/SED', cc_path='/home/robinl/ComboCode/Data')

source code 

Retrieve the photometry of a star through the IvS repo's SED builder.

Save the photometry in a target location.

Parameters:
  • star_name (str) - Name of the star (cc name from Star.dat)
  • fn (str) - Output filename of the photometry file. Always appends '_STAR.dat' where STAR is star_name. The file is saved in dp.

    (default: 'Photometric_IvS')

  • psuffix (str) - path suffix for dp where the Raw photometry as downloaded by the SED tool is saved. To re-download, delete the file associated with the star.

    (default: 'Raw/IvS_SEDTool/')

  • remove (list[str]) - Photometry to be removed from the output file. e.g. ['WISE','DENIS']

    (default: [])

  • dp (str) - Location for saving the clean photometry file used as input for CC.

    (default: ~/Data/SED/)

  • cc_path (string) - path to the combocode folder

    (default: ~/ComboCode/Data/)

getExtinctionCurve(gal_position='ism', curve_type='chiar_tielens', av_to_ak_conv=0.112, path='/home/robinl/MCMax/Extinction_Curves')

source code 

Read extinction curve.

Parameters:
  • gal_position (string) - galactic position star: 'ism' or 'gc'. Difference of extinction curve for ISM or galactic center (as for chiar_tielens), if no difference (as for cardelli) this parameter is ignored.

    (default: 'ism')

  • curve_type (string) - type of extinction curve ('chiar_tielens' or 'cardelli')

    (default: 'chiar_tielens')

  • av_to_ak_conv (float) - V-band to K-band conversion factor for extinction correction

    (default: 0.112)

  • path (string) - path for extinction curve data

    (default: '~/MCMax/Extinction_Curves/')

Returns: (array,array)
The wavelength grid and extinction curve values
