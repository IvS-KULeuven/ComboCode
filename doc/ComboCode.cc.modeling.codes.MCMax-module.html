readModelSpectrum(dpath, rt_sed=1, fn_spec='spectrum45.0.dat')

source code 

Read the model output spectrum.

If no ray-tracing is requested or no ray-tracing output is found, the average of the MC spectra is taken.

Parameters:
  • dpath (string) - folder that contains the MCMax outputfiles
  • rt_sed (bool) - If a ray-traced spectrum is requested

    (default: 1)

  • fn_spec (str) - The filename of the ray-traced spectrum. Typically this is the default name, but can be different depending on the ray-tracing angle that is used. Not used if MCSpec are used.

    (default: spectrum45.0.dat)

Returns: (array,array)
The wavelength and flux grids (micron,Jy)

Read the model output visibilities, either as function of wavelength or baseline.

Parameters:
  • dpath (string) - folder that contains the MCMax outputfiles
  • fn_spec (str) - The filename of the ray-traced visibilities. Typically this is the default name, but can be different depending on the inclination (or baseline) that is used.

    (default: visibility01.0.dat)

Returns: dict
A dictionary containing either wavelength or baseline, the flux, and the visibilities for either given baselines or wavelengths

rayTraceSpectrum(model_id, path_mcmax='runTestDec09', inputfilename='', redo_rt=0)

source code 

Do the ray-tracing of the spectrum according to ~/MCMax/Observation_Files/Spec.out, but only if spectrum45.0.dat does not exist yet.

Parameters:
  • model_id (string) - the model_id of the requested model
  • path_mcmax (string) - modeling folder in MCMax home
  • inputfilename (string) - the inputfilename of the model. if '': filename is inputMCMax_model_YYYY-MM-DDhHH-mm-ss.

    (default: '')

  • redo_rt (bool) - redo the ray tracing of the spectrum regardless of the spectrum already existing or not

    (default: 0)

rayTraceImage(model_id, path_mcmax='runTestDec09', inputfilename='', remove_source=0, output_folder='')

source code 

Do the ray-tracing of images, according to ~/MCMax/Observation_Files/Image.out.

Parameters:
  • model_id (string) - the model_id of the requested model
  • path_mcmax (string) - modeling folder in MCMax home
  • inputfilename (string) - the inputfilename of the model. if '': filename is inputMCMax_model_YYYY-MM-DDhHH-mm-ss.

    (default: '')

  • remove_source (bool) - remove the central source from the image

    (default: 0)

  • output_folder (str) - The location of the output folder. By default, set at the model output folder.

    (default: '')
