Initializing an instance of the Radiat class.

Parameters:
  • molecule (Molecule()) - The molecule (short hand name) for which radiat info is loaded.
  • use_indices_dat (bool) - If True, get the radiat filename from ~/ComboCode/Data/Indices.dat

    (default: 0)

  • path (string) - The path where the radiat file can be found. Uses radiat_backup appended to the path in case use_indices_dat is True.

    (default: ~/GASTRoNOoM/src/data/)
