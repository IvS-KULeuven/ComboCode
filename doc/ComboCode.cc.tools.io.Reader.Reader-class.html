readFile(self, filename, delimiter=' ')

source code 

Read a filename and store its contents in the OutputReader.

The contents are stored in a dictionary, with the filename as key.

The contents can be returned when running getFile.

Parameters:
  • filename (string) - The filename of the outputfile, including filepath
  • delimiter (string) - The delimiter character used in this file.

    (default: ' ')
